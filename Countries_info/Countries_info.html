How to use the countries info command line App

The Countries info commandline app is an app that provides country information from the Countries API The information provided ranges from:

  • Country Name
  • Country Top Level Domain
  • Country Capital
  • Country Region and Subregion
  • Country Population
  • Country GeoLocation
  • Country Area in sq Km
  • Country or countries it borders
  • Country currencies
  • Country Languages

This App has two parts

  • Simple Interface
  • Advanced Interface

The simple part of the app just returns results as they have been searched, e.g search by country returns all the information of the Country, but the Advanced part first takes a simple search parameter like a country name or currency and the refined searches can be performed on the results obtained e.g starting with all countries, refine by getting only countries which border tunisia and have languages like french
