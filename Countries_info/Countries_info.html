How to use the countries info command line App

The Countries info commandline app is an app that provides country information from the Countries API The information provided ranges from:

  • Country Name
  • Country Top Level Domain
  • Country Capital
  • Country Region and Subregion
  • Country Population
  • Country GeoLocation
  • Country Area in sq Km
  • Country or countries it borders
  • Country currencies
  • Country Languages

The code for this app is hosted on github

Usage:
Language: Python
Version: 3.xx

This App has two parts

  • Simple Interface
  • Advanced Interface

The simple part of the app just returns results as they have been searched as shown below there are choices to choose from;

                         Welcome to Country info Data Search
                         ###################################

                     Please select a Search criteria choice below:

                1. By country name
                2. By country code
                3. "all" for all countries
                4. By region
                5. By subregion
                6. By capital city
                7. By Calling code
                8. By ISO 639-1 Language e.g "en", for English


                Enter Your Choice: _
By Country

When you choose 1, you are prompted to enter the country name, Country names are the normal country names e.g Kenya, USA, Mexico or Uganda

By Country Code

When you choose 2, you are prompted to enter the country code, Country Codes are the normal country names e.g ke for Kenya, USA for USA, tz for Tanzania, ug for uganda, rus for Russia

All for all countries

When you choose 3, you are not prompted to enter anything since the default is 'all' to get all the data available for all the Countries

By Region

When you choose 4, you are not prompted to enter Region. The world is divided into regions, for any region entered all the countries in that region will be returned together wih their data, regions include, Africa, Asia, Americas, Europe n so forth

By SubRegion

When you choose 5, you are not prompted to enter SubRegion. The world is further divided into sub-regions, for any sub-region entered all the countries in that sub - region will be returned together wih their data, sub-regions include, Eastern - Africa, Western Asia, South America n so forth

By Country Capital City

When you choose 6, you are prompted to enter the country capital city, Capital cities are the normal capital city names e.g Nairobi for Kenya, New York for USA, Dar es salaam for Tanzania, Kampala for uganda

By Calling Code

When you choose 7, you are prompted to enter the calling code, calling codes are hose pre-appended to a number before calling, they identify the country of the person being called e.g 1 for USA, 254 for Kenya, 238 for cape verde, 236 for Congo

By ISO 639-1 Language

When you choose 8, you are prompted to enter the language, ISO 639-1 language is an abbreviation format, e.g en for English, aa for Afar, fr for french this codes can be found here
